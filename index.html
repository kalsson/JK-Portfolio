Habit Tracker

Habit Tracker is a C# console application designed to help users keep track of their daily habits efficiently. This project is perfect for those looking to perform basic CRUD (Create, Read, Update, Delete) operations on their habits in a simple and intuitive way. The application utilizes ADO.NET for database operations and stores data in an SQLite database.

Coding Tracker

Coding Tracker is a C# console application designed to help users log and track their coding sessions. This project is ideal for developers who want to monitor their coding habits and progress over time, offering simple and effective CRUD (Create, Read, Update, Delete) operations to manage their coding logs. The application employs Dapper for database access and stores data in a SQLite database.

Flashcards

Flashcards is a C# console application designed to help users create and manage digital flashcards for studying. This project is ideal for students and learners who want to enhance their memorization skills, offering simple and effective CRUD (Create, Read, Update, Delete) operations to handle their flashcards efficiently. The application leverages Dapper for ORM and Spectre.Console for a rich console interface. It uses a SQL Server database to store the flashcard data.

Drinks info

Drinks is a C# console application designed to help restaurant employees search for and manage drink recipes from an external vendor's database via an API. This project is ideal for drink enthusiasts who want easy access to recipes and the ability to manage their collection of drinks. The application uses REST APIs to fetch data about drinks from external sources and presents it in a user-friendly manner.

Phone Book

Phonebook is a C# console application designed to help users manage their contacts efficiently. This project is ideal for individuals who want to keep track of their contacts in a simple and organized manner. The application uses Entity Framework Core for database operations with SQL Server and presents data in a user-friendly manner using Spectre.Console.

Shifts Logger

The application uses Entity Framework Core for database operations with SQL Server and presents data in a user-friendly manner.

Sports Results Notifier

Sports Results Notifier is a C# console application designed to help users track and receive updates on sports results. The application uses HTML parsing to fetch sports results from a specific website and sends updates via email.

Excel Reader

Excel Reader is a C# console application designed to help users read and manage data from Excel files efficiently. This project is ideal for individuals who frequently work with Excel files and need a streamlined way to process and analyze data. The application uses EPPlus to handle Excel file operations and Entity Framework Core for database operations with SQLite.
